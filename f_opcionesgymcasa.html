Alquilar Equipos Para Entrenar en Casa

En esta sección podrás escoger los equipos* para ejercitarte en tu hogar como si tuvieras tu propio gimnasio!

Recuerda que podrás alquilar nuestros equipos por semanas o meses y usarlos en la comodidad de tu hogar o donde desees.

*Los equipos cuentan con una garantía que cubre el 90% de los daños. Los costos de transporte e instalación están incluidos.

El tiempo mínimo de alquiler son dos semanas. Para más info de las regulaciones aquí

Maquina multifuncional

Maquina multifuncional para espalda, biceps, triceps, y pierna.

Se le puede agregar peso para el entrenamiento.

Pesas no incluidas.

Valor: $50/semana.

Maquina de Remo

Esta maquina de remo permite hacer ejercicios para mejorar la resistencia y como buen cardio.

Es un buen aliado para fortelecer piernas, brazos, espalda, y reforzar el core.

La intensidad se puede graduar.

Valor: $40/Semana.

Maquina de presión

Perfecta para un entrenamiento extremo de piernas y pantorrilas.

Sirve para hacer sentadillas, prensa y diagonal.

4 pesas de 40kg incluidas.

Se puede agregar más pesas.

Soporta hasta 300Kg.

Se recomienda usarse bajo supervisión de otra persona.

Valor: $50/Semana.

Bicicleta para Spinning

Bicicleta para entrenamiento intensivo de Spinning.

Permite hacer ejericios de cardio, resitencia y fortaleza para las piernas.

Elemento ideal para ejercicios de entrenamiento para competencias ciclistas.

Incluye pantalla y con conexión bluetooth y holder para celular.

Soporta máximo 200Kg.

Valor: $45/Semana.

Bicicleta eliptica

Bicicleta eliptica para hacer cardio de resitencia.

Perfecta para mejorar habilidad y rapidez en las piernas.

Se recomienda especialmente para corredores, patinadores y jugadores de fútbol.

Perfecta para ejercitar piernas sin impacto en la rodilla.

Soporta máximo 200kg

Valor: $4/día-$12 Semana.

Maquina para caminar como cardio.

Maquina para entrenar marcha, running.

Graduación de velocidades.

Pantalla LED con conexión bluetooth.

Graduación de nivel para la banda.

Función de escaladora.
